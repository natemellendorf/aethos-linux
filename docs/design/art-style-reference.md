# Aethos Client Art Style Reference

This document captures the visual direction provided by the project owner for all Aethos clients.

## Core style direction

- Palette focus: deep-space blue + luminous purple.
- Aesthetic: spacecraft cockpit + translucent glass.
- Feel: calm, premium, high-contrast, futuristic.

## UI motifs to preserve

- Frosted/translucent glass cards with subtle white borders.
- Layered gradients over dark navy backgrounds.
- Soft glow highlights and bloom-like accents.
- Rounded corners with minimal hard edges.
- Clean typography with bright white headings and muted support text.

## MVP1 Linux implementation alignment

- Introduce dark blue base background gradient.
- Add translucent glass panel containers for content.
- Use blue-purple accent gradients for action buttons.
- Keep text contrast high for readability.
